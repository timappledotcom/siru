# Siru

Siru is a static site generator inspired by [Hugo](https://gohugo.io/), built with Ruby.

## Features
- Markdown content rendering
- Theme support
- Live server with reload

## Usage

1. **Create a new site**
   ```
   siru new [sitename]
   ```

2. **Build the site**
   ```
   cd [sitename]
   siru build
   ```

3. **Serve the site**
   ```
   siru serve
   ```
